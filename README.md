又是一个给自己写的小工具。

此工具使用全能护盾beta版本的UI

Twitter（或者说X）现在的评论区和转推列表，全是机器人、营销号、加密货币诈骗。你随便发一条推，底下立刻涌上来一群Bot，刷着同样的诈骗链接，用着同样的头像模板。

我受够了。

所以写了这个。

---

## 它能做什么

**关键词过滤**
- 支持云端规则库 + 本地关键词
- 命中关键词的推文直接隐藏
- 正则表达式？暂时没加，以后看情况

**用户黑名单**
- 可以把特定用户加入黑名单
- 他们的推文不会再出现在你的时间线里

**自动处刑（危险功能）**
- 自动扫描转推列表里的Bot
- 把它们批量加入黑名单
- ⚠️ **警告：这个功能会导致高频拉黑操作，可能被Twitter判定为滥用行为，账号有冻结风险。自行承担后果。**

**开发者模式**
- 不隐藏推文，只用红色虚线框标记
- 方便调试规则，看看哪些被误杀了

## 界面

点击右下角的硬币悬浮球，会有一个~~中二的~~超电磁炮发射动画，然后弹出控制面板。

配色是御坂美琴的风格——审判红 + 警告黄。

控制台会实时显示拦截日志。

---

## 安装

1. 先装一个用户脚本管理器
   - [Tampermonkey](https://www.tampermonkey.net/)（推荐）
   - [Violentmonkey](https://violentmonkey.github.io/)

2. 安装脚本
   - 直接点击 `.user.js` 文件安装
   - 或者从 Greasy Fork 安装（如果我有发布的话）

3. 打开 Twitter / X，右下角会出现一枚硬币

## 云端规则库

默认使用我自己维护的规则库：

https://raw.githubusercontent.com/Sakurairinaqwq/Twitter-Railgun/main/BlockTwitter.json

## 为什么叫 Railgun

因为我喜欢《某科学的超电磁炮》。

御坂美琴用硬币发射超电磁炮，我用代码发射净化光束。

都是小小的、个人的反抗。打不穿什么，也改变不了什么。但至少能让自己的时间线清净一点。
